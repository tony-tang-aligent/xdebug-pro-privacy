Privacy Policy – Aligent Xdebug Pro

Last updated: September 30, 2025

Overview

Aligent Xdebug Pro is a Chrome extension designed for PHP developers. Its sole purpose is to simplify and manage Xdebug session cookies for cross-domain debugging in modern Progressive Web App (PWA) and microservice environments.

Data Collection

This extension does not collect, store, or transmit any personal or sensitive user data. All configuration values (such as IDE keys, domains, and cookie settings) are stored locally on your device using Chrome’s storage API and are never shared externally.

Permissions

  • storage: Used only to save your debugging configuration (IDE key, domain, cookie options) locally.
  • cookies: Required to create, update, and remove the XDEBUG_SESSION cookie on specified domains.
  • host permissions: Required only to apply cookies to domains you configure (e.g., .aligent.dev, api.example.com).

These permissions are used exclusively for managing debugging sessions and are not used for tracking or unrelated activities.

Third Parties

The extension does not send data to any third parties and does not use remote code. All logic is self-contained within the extension package.

User Rights

Since no personal data is collected, there are no user data requests or deletions required. All extension-related data can be cleared at any time by removing stored settings via the extension interface or uninstalling the extension.
